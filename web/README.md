# BoB Web UI (SvelteKit)

This UI is embedded in Tauri for desktop use and is now focused on:

- starting/selecting conversations through `bob-api`
- sending messages via API
- syncing local chat cache via desktop commands
- managing local GUI/context/personalization profile

## Dev

1. Start backend service (`bob-api`)
2. `cd BOB_Web`
3. `npm install`
4. `npm run dev`
